### How to install/test
This app build with React, Express & MongoDB Atlas<br/>

WS (under lioenix-ws folder)<br/>
build: npm i <br/>
start: npm start<br/>
test: npm run dev<br/>

UI (under lioenix-ui folder)<br/>
UI build: npm i (under lioenix-ui folder)<br/>
UI start: npm start(under lioenix-ui folder)<br />
